Introduction

Please note that Aevan is still in development and is a constructed language (conlang) designed to be functional, secretive but still goofy. It follows simple rules of pronunciation and grammar while offering deep flexibility for self-expression. This guide will teach you everything you need to master Aevan!

  • Phonetics: Easy-to-pronounce vowels and consonants.
  • Word Order: Subject-Verb-Object (SVO).
  • Tenses: Prefix-based (ka- for past, va- for future).

Rules

Pronunciation:

  • Vowels: a (ah), e (eh), i (ee), o (oh), u (oo), y (ih)
  • Consonants: b, d, f, g, h, k, l, m, n, p, r, s, t, v, z

Word Order: Subject + Verb + Object (SVO).

Tenses:

  • Present: No prefix (e.g., sa = see).
  • Past: Prefix ka- (e.g., ka-sa = saw).
  • Future: Prefix va- (e.g., va-sa = will see).

Negation: Add na- before a verb (e.g., na-sa = donâ€™t see).

Questions: Add -la to the end of a sentence (e.g., Ni sa zu-la? = Do I see you?).

Grammar

Aevan grammar is straightforward:

  • Adjectives follow nouns (e.g., Tari vilo = Happy friend).
  • Possession uses tyva (e.g., Kiva tyva vara = Book of the sky).
  • Comparisons use taka (e.g., Zu taka sira = You are like a star).

Common Phrases

  • Hello: Vura zu! (Light to you!)
  • Goodbye: Vyn nari. (Into the night/home.)
  • Thank you: Tava luva. (Have heart.)
  • Sorry: Ni na-mira. (I did not think.)
